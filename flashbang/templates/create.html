So you want to create your own randomised flashcards; you've come to the right place!

First things first, you'll need go to Google Spreadsheet and create a new spreadsheet. This is where the flashcards will draw all the questions/answers from.

The first row should act as a header (so we know what each column is supposed to represent) and the first column should be where the questions go. You can have as many columns as you like but only the first column will be initially visable in the flashcards, the other columns will be considdered answer columns and the user will need to roll over or highlight them to see the answer.

When you're happy with the spreadsheet, click File > Publish to the web.

This will open a dialog with two drop down menus, select your sheet from the first drop down and "Comma-separated values (.csv" from the second and click "Publish". You should get a pop up asking "Are you sure you want to publish this selection?", click "ok".

When you've done this, the dialog will change. A link will appear asking you to copy it with Ctrl+v and paste it in the form below along with a memorable name and click "Submit".

At this point, you'll be redirected to your new flashcards.
